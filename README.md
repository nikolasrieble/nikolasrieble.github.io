# nikolasrieble.github.io

Personal website and blog.

## Structure

- `index.html` - About me page
- `blog.html` - Blog index with list of posts
- `blog/post.html` - Dynamic blog post renderer (loads markdown files)
- `blog/*.md` - Blog posts in markdown format
- `styles.css` - Global styles

## Local Development

Open `index.html` in a browser to view the site locally.

## Deployment

This site is automatically deployed via GitHub Pages from the `main` branch.
